# Slides

The live deck is rendered by the optional React app from [`src/data/slides.tsx`](../src/data/slides.tsx) and shown in the **Presentation** tab (`npm run dev`).

Keeping the deck as code means it stays in sync with the repo (same examples, same prompts) and renders consistently in light and dark. Export to `.pptx` only if a venue requires a static file.
